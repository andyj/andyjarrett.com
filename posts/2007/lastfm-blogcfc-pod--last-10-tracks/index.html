Last.fm blogcfc pod - last 10 tracks v0.5

Since moving over to the Mac I haven't had a "Now Playing" on the blog due to licensing issues with the software I was using (new OS = new license). Luckily last.fm offer a free feed of your last ten tracks (plus other stuff). You can see the pod in action if you look left now! Below is the pod code for BlogCFC, of course you can just rip the code out and use it for any other CF blog/site. For people that want to rip the code to their own blogs i have add the variable applicaiton._lastFM.feed['feedConsumed'] Using your application scope it will store the feed. For BlogCFC users this is not the case and will get a new copy of the feed EVERY HIT. At the moment I don't know how to do application scope caching per pod. (I think I most of had a full brain lobotomy or something). If anyone knows how then leave a comment and I'll have a dot release #applicaiton._lastFM.feed['feedConsumed']# #now()# #song['title'].xmltext#
